# 患者調査 Vignette + DCE/DCM モック

## 目的

患者調査は以下の4点を明らかにするための設計案である。

1. 外来導入レジメンの内容について患者の意向を知り、レジメン改善に活かす。
2. 入院導入なら嫌だが、外来導入なら受けたい、または前向きに考えたい患者がどの程度いるかを明らかにする。
3. 来年度の安全性検証研究の潜在対象者リクルートにつなげる。
4. 患者のニーズや受容性と医師判断のギャップを、クロザピン導入の意思決定起点となることの多い医師にインフォームする。

## 方法論

- Vignette比較: 入院導入、外来導入週2回、外来導入週1回、外来導入週1回+訪問看護を同じ5段階で評価する。
- DCE/DCM: 外来導入レジメン属性のトレードオフを離散選択課題で評価する。
- 障壁・促進因子: レジメン案とコメントから、週2回通院、毎日チェック、支援者確認、発熱時対応、異常時入院切替、便秘、眠気、体重増加などを評価する。
- リクルート意向: 安全性検証研究の説明希望、条件が合えば参加検討、主治医と相談したいかを聞く。

## 対象集団

- Primary sample: クロザピン適応相当または治療抵抗性が疑われる外来統合失調症スペクトラム障害患者のうち、クロザピン未使用者。
- Secondary/preference sample: 外来通院中の統合失調症スペクトラム障害患者のうち、クロザピン未使用者。
- 除外: 現在クロザピン使用中、過去にクロザピン使用歴あり、同意能力が不十分、主治医が調査依頼不適当と判断。
